กสทช. เตรียมจัดการประมูลทีวีดิจิตอล ปลายเดือนตุลาคมนี้  เล็ง แจกคูปองส่วนลดซื้อ set top box เดือนหน้า

พ.อ.นที ศุกลรัตน์   รองประธาน กสทช. เปิดเผยความคืบหน้าของการเตรียมประมูลทีวีดิจิตอล ว่า จะจัดการประมูลได้ช่วงปลายเดือนตุลาคมถึงต้นเดือนพฤศจิกายน และเริ่มออกอากาศอย่างเป็นทางการได้ปลายปีนี้ ซึ่งเบื้องต้นจะสามารถรับชมทีวีดิจิตอลได้ถึง 30 ช่อง ในกรุงเทพฯ ปริมณฑล และจังหวัดสำคัญ ก่อนเร่งขยายให้ครอบคลุมทั่วประเทศ
ส่วนการแจกคูปองส่วนลด ให้กับประชาชนนำไปเป็นส่วนลดซื้อโทรทัศน์ระบบดิจิตอล หรือ set top box เพื่อรับสัญญาณทีวีดิจิตอลสำหรับเครื่องรับโทรทัศน์รุ่นเก่าจะเริ่มดำเนินการทันทีหลังการประมูลแล้วเสร็จ และจะอนุญาตให้ขาย set top box อย่างเป็นทางการในเดือนสิงหาคมนี้
