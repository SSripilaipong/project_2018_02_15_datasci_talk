หุ้นไทยเปิดตลาด (17 เม.ย.)บวก 1.61 จุด ซึ่งดัชนีอยู่ที่ 1,403.45 จุด มูลค่าซื้อขาย 1,054.96 จุด

บล.ทิสโก้มองดัชนีหุ้น (SET Index) มีโอกาสเดินหน้าขึ้นต่อในช่วงครึ่งเดือนหลัง จากการเมืองที่เริ่มเห็นเค้าลางของทางออก และการซื้อต่อเนื่องของต่างชาติ แต่การดีดตัวขึ้นน่าจะจำกัดแถว 1420จุด เรามองเป็นเป้าหมายขายทำกำไรระยะสั้น เพราะ SET Index ระดับดังกล่าว นอกจากจะเป็นเป้าหมายในปีนี้ของเราตามปัจจัยพื้นฐานแล้ว  ยังเป็นกรอบดัชีสูงสุด (แนวต้าน)ที่สำคัญทางปัจจัยเทคนิคอีกด้วย เราแนะนำให้เริ่มทยอยขายทำกำไรเมื่อ SET Index เข้าใกล้ระดับดังกล่าว เพราะเราเชื่อว่า SET Index ยังไม่สามารถยืนเหนือ 1400 จุดได้อย่างมั่นคง เพราะประมาณการเศรษฐกิจและผลกำไรของบริษัทจดทะเบียนยังมีความเสี่ยงปรับลงอยู่ และเราเป็นห่วงการกลับมาใช้ความรุนแรงหลังทราบผลชี้ชะตาทางการเมืองผ่านไปสักระยะหนึ่งแล้ว
หุ้นเด่น สำหรับเดือน เม.ย. ของบล.ทิสโก้ ยังคงเป็น ADVANC, BBL, BCP, BGH, HMPRO, ITD, IVL, PYLON หลัก ๆ เนื่องจาก (1) เป็นหุ้นพื้นฐานดีขนาดใหญ่ที่น่าจะเป็นเป้าซื้อคืนของต่างชาติ และ (2) หุ้นรับเหมาฯ ที่ซึบซับข่าวร้ายทางการเมืองไปมากแล้ว เราเพิ่มธีมหุ้นน่าลงทุนใหม่ คือ หุ้นที่มีแนวโน้มกำไรดีอยู่ กลุ่มสื่อสาร  JAS, กลุ่มวัสดุก่อสร้าง  SCC, กลุ่มอาหาร TUF, TVO
