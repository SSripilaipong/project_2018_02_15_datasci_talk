“ตลาดสัญญาซื้อขายล่วงหน้า” ลดไซส์ SET 50 Index Futures เป็น MINI SET 50 Index Futures เพื่อเพิ่มปริมาณการซื้อขาย  “เกศรา” คาดปริมาณซื้อขายเฉลี่ยต่อวันเหลือ 5.5 หมื่นสัญญา จากปีก่อน 7.5 หมื่น เหตุภาวะการเมือง

นางเกศรา มัญชุศรี รองผู้จัดการ หัวหน้าสายงานการตลาด ตลาดหลักทรัพย์แห่งประเทศไทย เปิดเผยว่า ตลท.ประกาศใช้ สัญญาชื้อขายล่วงหน้าแบบใหม่คือ MINI SET 50 จะมีมาร์จิ้นที่เพิ่มมากขึ้นจากเดิมที่มีอยู่ ซึ่งในส่วนของ SET 50 เดิมนั้นจะมีมาร์จิ้นอยู่ที่ประมาณ 2 หมื่นบาท/สัญญา โดยการคิดค่าธรรมเนียม MINI SET 50 ในรูปแบบใหม่นั้นจะปรับลดค่าธรรมเนียมลง (หาร 5 จากค่าเฉลี่ยต่อสัญญาในรูปแบบเดิม) ซึ่งจะมีค่าธรรมเนียมเทียบเท่าการซื้อขายแบบประเภทออปชัน (สัญญาที่กำหนดสิทธิ์ ว่าเป็นผู้ซื้อ หรือผู้ขาย ณ เวลาที่กำหนด)
ทั้งนี้ ทางตลาดหลักทรัพย์แห่งประเทศไทย จึงได้พิจารณาปรับปรุง SET 50 Index Futures โดยได้ปรับเปลี่ยนเรียกเก็บค่าธรรมเนียมแบบใหม่ ในรูปแบบของ MINI SET 50 Index Futures ที่จะมีขนาดของสัญญา และราคาที่ถูกลง ซึ่งจะทยอยยกเลิก SET50 Index Futures ระบบเก่าเมื่อสัญญาถึงกำหนดครบรอบหมดอายุลง และเปลี่ยนมาใช้สัญญาซื้อขายล่วงหน้าแบบใหม่เป็น MINI SET 50 Index Futures แทนทั้งหมด โดยจะเริ่มทำการซื้อขายในวันที่ 6 พ.ค. เป็นวันแรก ขณะเดียวกัน ตลาดสัญญาซื้อขายล่วงหน้า หรือ TFEX จะปรับเปลี่ยนเวลาเปิดตลาดซื้อขายภาคบ่ายใหม่ที่เวลา 14.15 น ในวันเดียวกัน
อย่างไรก็ดี แนวโน้มประมาณการซื้อขายสัญญาล่วงหน้าคาดว่าจะเฉลี่ยอยู่ที่ประมาณ 5.5 หมื่นสัญญา/วัน จากเป้าเดิมที่ตั้งไว้ที่ประมาณ 7.5 หมื่นสัญญา/วัน ส่วนสาเหตุที่ปรับลดเป้าลงมานั้น ส่วนหนึ่งมาจากปัจจัยด้านความขัดแย้งทางการเมืองที่มีความรุนแรง และกระทบความเชื่อมั่นนักลงทุน และปัจจัยลบจากมาตรการ QE ของสหรัฐอเมริกา ที่ทยอยลดวงเงินจัดซื้อพันธบัตรลงเหลือเพียง 6.5 หมื่นล้านเหรียญ/เดือน ซึ่งได้ประกาศเริ่มดำเนินการแล้ว
