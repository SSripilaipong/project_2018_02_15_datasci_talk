หุ้นเปิดตลาด ( 6พ.ค.)บวก 2.43 จุด หรือ 0.17% ดัชนีอยู่ที่ 1,423.91 จุด มูลค่าซื้อขาย 768.17 ล้านบาท การเมืองเริ่มร้อนแรงอาจกดดันตลาด ต่างชาติยังเล่นหุ้นไทย

นายปริญทร์ กิจจาทรพิทักษ์ นักวิเคราะห์หลักทรัพย์ บล.เคทีบี (ประเทศไทย) เผยการเคลื่อนไหวของดัชนีหุ้น SET Index เมื่อวันศุกร์ (2 พ.ค.57) แม้มีความผันผวนจากการแกว่งตัวของดัชนีตลาดหุ้นไทยตลอดวันทำการ ดูเหมือนว่า นักลงทุนในตลาดยังให้การตอบรับทางบวกจากปัจจัยต่างประเทศ เพราะSET Index ยังเคลื่อนไหวในแดนบวกทั้งวัน วันนี้หุ้นเปิดตลาด ( 6พ.ค.)บวก 2.43 จุด หรือ0.17% ดัชนีอยู่ที่ 1,423.91 จุด มูลค่าซื้อขาย 768.17 ล้านบาท
บล.เคทีบี มองแนวโน้มแกว่งตัวขึ้นทดสอบแนวต้านเดิมบริเวณ 1,425-1,430 จุด ซึ่งไม่ใช่ระยะทางที่ไกลมากนักจากระดับดัชนีตลาดล่าสุด อย่างไรก็ตาม ตลาดยังถูกกดดันโดยปัจจัยการเมืองในประเทศ แม้ยังไม่มีความรุนแรงแต่อย่างใด ขณะที่แรงซื้อที่ต่อเนื่องของนักลงทุนต่างชาติกำลังจะทำให้สถานะผู้ขายสุทธิเปลี่ยนแปลงไปเป็นผู้ซื้อสุทธิ หากยังต่อเนื่องอีกเพียงไม่กี่วันเท่านั้น ซึ่งเรามองว่า โอกาสที่ตลาดหุ้นไทยจะปรับขึ้นยังมีอยู่ ผนวกกับแรงเก็งกำไรหุ้นรายตัวที่มีผลประกอบการงวด 1Q57 ออกมาเชิงบวก ยังเป็นปัจจัยหนุนราคาหุ้นเฉพาะตัวที่ดีในช่วงนี้ เราให้กรอบการเคลื่อนไหวของ SET วันนี้ในช่วง 1,415-1,425 จุด ไม่เกิน 1,430 จุด บล.เคทีบี มองว่าการแกว่งตัวจะแรงขึ้นเมื่อปรับตัวขึ้นมาที่บริเวณนี้
หุ้นเด่นวันนี้ แนะนำขายทำกำไรระยะสั้น MBKET-ASP-ZMICO แต่ซื้อเก็งกำไร CPALL-PTT-TUF เราแนะนำให้นักลงทุนหลีกเลี่ยงหุ้นโบรกเกอร์ในช่วงครึ่งปีแรก 1ปี 2557 เนื่องจากชัดเจนว่า ผลประกอบการทุกบริษัทจะได้รับผลกระทบจากมูลค่าการซื้อขายที่หดตัวให้น้ำหนักที่ดีจากหลักเกณฑ์ดังกล่าวกับ CPALL-PTT-TUF เราให้ราคาเป้าหมาย CPALL ที่ 55.60 บาท, PTT ที่ 420 บาท และTUF ที่ 75 บาท
