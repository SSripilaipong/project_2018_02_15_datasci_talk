โบรกฯคาดหุ้นไทย วันนี้ยืนเหนือ 1,600 จุด จากปัญจัยบวกหนุน และจิตวิทยาลงทุนเป็นบวก
บริษัท หลักทรัพย์ เคเคเทรด  เปิดเผยว่า  SET  มีโอกาสปรับขึ้นต่อจากจิตวิทยาลงทุนเป็นบวก (1) ยอดซื้อสุทธินักลงทุนต่างชาติและสถาบันรวมกันกว่า 8 พันล้านบาท
โดยนักลงทุนต่างชาติซื้อสุทธิ เป็นวันที่ 3 ติดต่อกัน (2) ความคาดหวังการปรับลดดอกเบี้ยนโยบายหลังการ ประชุม กนง.  ในวันที่ 28 ม.ค. เพื่อกดไม่ให้เงินบาทแข็งค่ามากเกินไปจนอาจกระทบต่อการส่งออก
ซึ่ง หุ้นกลุ่มธนาคาร จะได้ประโยชน์ชัดเจนจากอัตราดอกเบี้ยเงินฝากที่จะลดลงเร็วกว่าเงินกู้ (3)  ราคาน้ำมันดิบ เริ่มทรงตัว จะลดแรงกดดันหุ้นกลุ่มพลังงาน และ (4) BAY ปรับขึ้น Ceiling เมื่อวันศุกร์มีผลต่อ  SET  ราว 16 จุด
หากราคาหุ้น BAY ยังปรับขึ้นต่อในวันนี้ก็จะเป็นผลบวกต่อ  SET  ในทางเทคนิค  SET  เพิ่งกลับสู่ทิศทางขาขึ้น มีแนวต้านแรกบริเวณ 1603 จุด ซึ่งเป็นจุดสูงสุดเดิมผ่านขึ้นไปได้จะไปทดสอบ 1650 (123.6% Fibonacci) ผ่านไม่ได้มีโอกาสลงมาปิด Gap แถว 1570 จุด
แนวโน้มวันนี้เป็นบวก มองกรอบเคลื่อนไหว 1,603 – 1,615 จุด
กลยุทธ์การลงทุน  เก็งกำไรระยะ 1-2 วัน ยืนเหนือ 1603 จุดถือครองหุ้นต่อ หรือหาจังหวะเก็งกำไร หาก SET ปิดต่ำากว่า 1603 จุด ชะลอการลงทุน
