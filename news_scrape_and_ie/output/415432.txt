โบรกฯ คาดเเนวโน้มตลาดหุ้นไทยวันนี้ พักตัวเเนะระวังเเรงขาย BAY หลังบริษัทออกมาปฏิเสธข่าว tender offer
บล.เคเคเทรด  เปิดเผยว่า แนวโน้มวันนี้ SET พักตัวอยู่ในกรอบ 1,570 – 1,600 จุด โดยมีจุดที่ต้องระวัง คือ แรงขายหุ้น  BAY  โดยเรามองว่า  BAY  มีความเสี่ยงทางขาลงหลังจากบริษัทออกมาปฏิเสธข่าวการทำ  tender offer
เพื่อออกจากตลาดเมื่อวานนี้มีผลทำให้ SET ปรับตัวลงและปิดลบ 10 จุด (ราคาหุ้น BAY ปรับลง 13% มีผลต่อ SET ถึง 9 จุด) ในขณะที่หุ้นธนาคารขนาดใหญ่อื่นๆยังแค่พักตัว
ดังนั้นหาก SET วันนี้ยังเหวี่ยงลงไม่ต่ำกว่า Gap แถวบริเวณ 1570 จุด เรายังคงมีมุมมองว่า SET ยังเป็นขาขึ้นในระยะสัปดาห์ โดยจะไปทดสอบ 1600 จุด และ 1650 จุด
จากโมเมนตัมขาขึ้นมาตรการ QE จาก ECB ซึ่งได้ผลักดันการเหวี่ยงตัวขึ้นของหลายๆตลาดหุ้นโลกอย่าง สหรัฐ ญี่ปุ่น ฮ่องกง และ เยอร์มัน โดยในตลาดหุ้นไทยมียอดซื้อสุทธิต่อเนื่องของนักลงทุนต่างชาติเป็นวานนี้เป็นวันที่ 4 ราว 2 พันล้านบาท
สำหรับวันนี้ มีปัจจัยให้ติดตาม คือ  การประกาศตัวเลขส่งออกและนำเข้าเดือน ธ.ค.  แม้เรามองว่าจะไม่มีนัยต่อตลาดนักเพราะคาดว่าตลาดรับรู้การไม่ฟื้นตัวของภาคส่งออกในปี 57 ไปแล้วยกเว้นว่าเห็นการฟื้นตัวโดดเด่นของรายนำเข้าสินค้าทุนจะถือเป็นการหนุนภาพการฟื้นตัวในเดือน ม.ค. ได้
กลยุทธ์การลงทุน ระยะ 1-2 วัน ยืนเหนือ 1570 จุดถือครองหุ้นต่อหรือหาจังหวะ “อ่อนตัวซื้อ” หาก SET ปิดต่ำกว่าให้ชะลอการลงทุน
