โบรกฯคาด  หุ้นไทย  ยังยืนเหนือ 1,600 ชี้!  กลุ่มพลังงาน นำโด่ง แต่มีโอกาสเปิดลบตามตลาดหุ้น ภูมิภาคยังหวั่นเฟดขึ้นดอกเบี้ย !!
บล.เคเคเทรด  เปิดเผยว่า แนวโน้ม  SET  วันนี้เป็นบวก มองกรอบเคลื่อนไหว  1600–1619  จุด SET มีโอกาสเปิดลบตามตลาดหุ้นภูมิภาคจากความวิตกเฟดจะปรับขึ้นดอกเบี้ยภายในปีนี้กลับเข้ามาอีกครั้ง
หลังสหรัฐประกาศตัวเลขเลขการจ้างงานเมื่อวันศุกร์ดีเกินคาด แต่โดยภาพรวมยังมองโอกาสแกว่งตัวขึ้นทดสอบจุดสูงสุดเดิมที่ 1619 จุด นักลงทุนต่างชาติและสถาบันกลับมาซื้อสุทธิพร้อมกันกว่า 3.3 พันล้านบาทเมื่อวันศุกร์ยังคงหนุนจิตวิทยาการลงทุนภาพรวม
ประกอบกับแรงเก็งกำไรในหุ้น กลุ่มพลังงาน จะกลับเข้ามาอีกหลัง ราคาน้ำมันดิบ Brent  ปรับขึ้นต่อเนื่องเป็นสัปดาห์ที่สองติดต่อกันจนล่าสุดทำจุดสูงสุดใหม่ในรอบห้าสัปดาห์ที่ US$56 ต่อบาร์เรล ทำให้ความคาดหวังราคาน้ำมันดิบผ่านจุดต่ำสุดไปแล้วเริ่มชัดเจน
กลุ่มธนาคาร คาดแรงเก็งกำไรกลับเข้ามาจากการคาดการณ์เศรษฐกิจฟื้นตัวรวมถึงโอกาสปรับลดดอกเบื้ยนโยบาย  อย่างไรก็ตาม หุ้นกลุ่มสื่อสารที่จะมีการทยอยประกาศงบปี 57 ในสัปดาห์นี้  (DTAC THCOM JAS INTUCH)
จะทำให้หุ้นกลุ่มนี้ยังคงผันผวนไร้ทิศทาง หลัง  ADVANC  ประกาศงบเมื่อวันศุกร์เป็นไปตามคาด  ภาพทางเทคนิคเป็นบวก หลัง  SET  ปิดเหนือบริเวณ 1600 จุด
และปรับขึ้นต่อเนื่องในวันศุกร์ จึงมีโอกาสขึ้นไปทดสอบ 1650จุดในระดับสัปดาห์ กลยุทธ์การลงทุน ระยะ 1-2 วัน หาก  SET  อ่อนตัวลงมาหาจังหวะเข้าลงทุน ไม่ไล่ราคา ปิดเหนือ 1610 จุดถือรอขาย
