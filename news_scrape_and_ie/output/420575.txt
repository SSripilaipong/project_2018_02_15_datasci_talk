โบรกฯชี้!  ดัชนีตลาดหุ้นไทยวันนี้ ยังผัวผวนไร้ทิศทาง –  กลุ่มพลังงาน ยังถ่วง!  กลยุทธลงทุน 1-2 วันนี้ หากดัชนีปิดเหนือ 1600 จุด เเนะนำถือเงินสด
บล.เคเคเทรด  คาด  SET  วันนี้มีโอกาสผันผวนไร้ทิศทางแถว 1600 จุดและคาดจะยังไม่ผ่านขึ้นไปโดยง่าย โดย หุ้นกลุ่มพลังงาน จะถ่วง  SET  หลังราคาน้ำมันดิบมีแนวโน้มกลับมาอ่อนตัวอีกครั้ง
โดย  Brent  ปรับลงมาแถว US$ 53 ต่อบาร์เรลลดลง US$2จาก2 วันก่อนหน้าหลังรายงาน IEA เปิดเผยสต๊อกน้ำมัดิบยังปรับขึ้นสู่ระดับสูงสุดเป็นประวัติการณ์ 417.9 ล้านบาร์เรลสูงกว่าที่นักวิเคราะห์คาด

ขณะที่การเจรจาเรื่องประนอมหนี้ของกรีซกับกลุ่มเจ้าหนี้ยุโรปยังไม่มีความคืบหน้าใดๆและอาจยืดเยื้อไปจนถึงปลายเดือนนี้ ทำให้ตลาดจะกลับมาให้น้ำหนักกับปัจจัยในประเทศอีกครั้ง
เรามองว่า  SET  จะคงผันผวนตามความคาดหวังต่อการประกาศผลประกอบการปี 57 ของบริษัทจดทะเบียนภายในเดือนนี้ และนัก 
ลงทุนจะเริ่มมองหาโอกาส สะสมหุ้นบางตัวที่คาดว่าผลประกอบการปี 58 จะเติบโตต่อเนื่อง
การปรับลงของ  SET  จะจำกัดและมีโอกาสรีบาวน์ได้ทุกเมื่อขณะที่ภาพทางเทคนิคยังไม่ชัดเจน แม้  SET  กลับขึ้นมาปิดเหนือ 1600 จุดแต่กราฟราย 10 นาที บ่งบอกถึงฟื้นตัวอย่างรวดเร็วในช่วงท้ายตลาดทำให้มีโอกาสขายทำกำไรระยะสั้น
หากคงปิดเหนือ 1600 จุดได้ในวันนี้จะแสดงถึงการฟื้นตัว   กลยุทธ์การลงทุน  ระยะ 1-2 วัน หาก  SET  เปิดกระโดดควรเป็นจังหวะขายทำกำไร
หรือหาจังหวะเข้าซื้อเมื่ออ่อนตัวเพื่อไปรอขายตอนรีบาวน์และ SET ปิดเหนือ 1600 จุดไม่ได้ให้กลับมาถือครองเงินสด แนวโน้มวันนี้เป็นกลาง มองกรอบเคลื่อนไหว 1,595 – 1610 จุด
