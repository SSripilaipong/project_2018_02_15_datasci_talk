หุ้นไทยวันนี้  ลุ้น!  จีดีพี Q4/57  แจ่ม คาดกลับยืน 1620 จุด ระบุกลุ่มพลังงานยังหนุน จากราคาน้ำมันดิบ สูงสุดในรอบ 7 สัปดาห์
บล.เคเคเทรด  คาด SET วันนี้มีโอกาสแกว่งตัวขึ้นทดสอบ 1620 จุดอีกครั้ง โดยกลุ่มพลังงานจะหนุนการปรับขึ้นของ SET หลังราคาน้ำมันดิบ Brent กลับมายืนเหนือบริเวณ US$60 ต่อบาร์เรลระดับสูงสุดในรอบ 7 สัปดาห์ได้อีกครั้ง
ขณะที่การประกาศตัวเลข  GDP  ไทย 4Q57 ในวันนี้หากออกมาดี (+2.5% ขึ้นไป) จะหนุนการฟื้นตัวต่อเนื่องของ SET แต่คาดว่าหุ้นรายตัวยังคงผันผวนตามการคาดหวังทั้งบวกและลบต่อการทยอยประกาศผลประกอบการปี 57


ที่จะเสร็จสิ้นภายในเดือนนี้ ส่วนอย่างไรก็ตามอัตราดอกเบี้ยพันธบัตรสหรัฐอายุ 10 ปี ที่ปรับขึ้นรวดเร็วมาทดสอบ 2% บ่งบอกถึงความผันผวนของตลาดการเงิน โอกาสที่จะเห็นเม็ดเงินจากนักลงทุนต่างชาติไหลเข้ามาในตลาดหุ้นช่วงนี้จะจำกัด
การเคลื่อนไหวของ SET จะขึ้นกับความคาดหวังเชิงบวกต่อปัจจัยในประเทศโดยเฉพาะด้านผลประกอบการ ซึ่งเราคาดว่าคงรอให้บริษัทจดทะบียนทยอยประกาศผลประกอบการปี 57 เสร็จสิ้น SET จึงจะเริ่มมีแนวโน้มที่ชัดเจน
การปรับขึ้นของ SET ยังคงผันผวนและมีโอกาสพลิกผันได้ทุกเมื่อ ส่วน  กลยุทธ์การลงทุน  ระยะ 1-2 วัน หาจังหวะเข้าลงทุนเน้นขึ้นขาย-ลงซื้อ และหาก SET ปิดต่ำกว่า 1610 จุดกลับมาถือครองเงินสด มองกรอบเคลื่อนไหว 1,610 – 1620 จุด
