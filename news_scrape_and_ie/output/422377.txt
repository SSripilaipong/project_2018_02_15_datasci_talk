โบรกฯ  คาดหุ้นไทยวันนี้สร้างฐานใต้ 1,600 จุด  เคลื่อนไหวตามปัจจัยในประเทศ ส่วนตลาดหุ้น จีน ไต้หวัน เกาหลีใต้ ปิดทำการวันตรุษจีน
บล.เคเคเทรด  เผย  SET  มีโอกาสปรับลงเพื่อสร้างฐานใต้ 1600 จุดอีกครั้งหลังตลาดขาดปัจจัยบวกใหม่ ขณะที่ความวิตกการปรับขึ้นดอกเบี้ยของสหรัฐยังคงกดดันตลาด ล่าสุดอัตราดอกเบี้ยพันธบัตรสหรัฐอายุ 10 ปียังคงปรับขึ้นมาที่ 2.1%
เป็นระดับสูงสุดในรอบ 8 สัปดาห์ไม่ดึงดูดเงินให้ไหลเข้ามาในภูมิภาค สะท้อนมายังการขายทำกำไรของนักลงทุนต่างชาติเป็นวันที่ 3 ติดต่อกันกว่า 2.1 พันล้านบาท และจะคงกดดันจิตวิทยาการลงทุน
วันนี้ ตลาดหุ้นภูมิภาค ทั้ง  จีน ไต้หวัน และเกาหลีใต้  ปิดทำการเนื่องใน  เทศกาลตรุษจีน  SET น่าเคลื่อนไหวตามปัจจัยในประเทศและปัจจัยเฉพาะของหุ้น

ซึ่งล่าสุดข่าวที่โฆษกของธปท.ออกมาย้ำจุดยืนในการดำเนินนโยบายการเงินโดยกล่าวว่า การลดดอกเบี้ยนโยบายไม่ช่วยกระตุ้นเศรษฐกิจและเงินบาทอ่อนค่า
ส่งสัญญาณอาจไม่ปรับลดดอกเบี้ยนโยบายตามข้อเสนอ สศช.ก่อนหน้านี้จะเป็นลบต่อความคาดหวังของตลาด อย่างไรก็ตาม เชื่อว่ากลุ่มพลังงานน่าจะมี downside จำกัด และมีโอกาสรีบาวน์ระหว่างวันหลังราคาน้ำมันดิบ Brent
ยังสามารถทรงตัวเหนือ US$62 ต่อบาร์เรล ส่วนหุ้นรายตัวจะยังผันผวนตามการคาดหวังทั้งบวกและลบต่อการทยอยประกาศผลประกอบการปี 57 ที่จะเสร็จสิ้นภายในเดือนนี้
ทางเทคนิคเกิดสัญญาณขายเมื่อ SET หลุดแนวรับบริเวณ 1600 จุดลงมา จึงมีโอกาสลงมาสร้างฐานใต้ระดับนี้อีกสักระยะ โดยมีแนวรับสำคัญบริเวณ 1560 จุด (Gap ที่เปิดทิ้งไว้ และเป็นระดับ 76.4% Fibonacci)
อย่างไรก็ตามหาก SET สามารถรีบาวน์กลับขึ้นปิดเหนือบริเวณ 1600 จุดได้จะยกเลิกสัญญาณขาย
กลยุทธ์การลงทุน  ระยะ 1-2 วัน ชะลอการลงทุนรอสัญญาณซื้อเมื่อ SET ยืนเหนือ 1600 จุดได้ หรือหาก SET ปรับตัวลงต่อ แนะนำเสี่ยงซื้อหุ้นกลุ่มพลังงานบางตัว กลุ่มปันผลเด่น และกลุ่มที่ลงแรงเมื่อวานนี้เพื่อไปขายช่วงรีบาวน์
