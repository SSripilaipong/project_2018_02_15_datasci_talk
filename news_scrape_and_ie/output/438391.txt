หุ้นไทยวันนี้ซึมต่อ โบรกฯ เเนะ’อ่อนตัวซื้อ’ มองแกว่งตัว กรอบ 1,560 – 1,575 จุด
บล.เคเคเทรด  มอง SET วันนี้แกว่งตัวสร้างฐาน โดยวานนี้นักลงทุนต่างชาติเสียโมเมนตัม การเป็นผู้ซื้อสุทธิ ซึ่งเป็นไปตามที่เราคาดหลังจากที่ราคาน้ำมันดิบเริ่มพักการปรับตัวขึ้นลดแรงหนุนจากกลุ่มพลังงานลง

ทางด้าน กลุ่มธนาคาร  งบการเงิน 1Q/58 ของธนาคารขนาดใหญ่อย่าง KBANK – SCB ประกาศแล้วพบว่าธุรกรรมที่เกี่ยวข้องกับธนาคารทั้งด้านการปล่อยสินเชื่อ NPL และ NIM ยังคงอ่อนแอ
มีเพียงแต่รายได้ค่าธรรมเนียมนายหน้าประกันที่ดีกว่าคาด รวมถึงธนาคารขนาดเล็กที่ประกาศไปแล้ว (TMB – TISCO – TCAP) ล้วนมีแนวโน้มแย่กว่าที่ Bloomberg consensus คาดเล็กน้อย
จึงยังคงมุมมองว่า BBL- BAY- KTB ที่จะประกาศหลังจากนี้ย่อมจะหนีไม่พ้นแนวโน้มดังกล่าวและจะส่งผลให้ SET มีโอกาสซึมต่อในระยะนี้
เรามอง Downside ในวันนี้ไม่มากนักจากปัจจัยบวกที่ธนาคารกลางจีนลดเกณฑ์กันสำรองเงินทุนของธนาคารพาณิชย์ (RRR) หวังกระตุ้นเศรษฐกิจ ซึ่งก็เห็นสัญญาณจากตลาดหุ้นภูมิภาคโดยเฉพาะยุโรปและสหรัฐหยุดการปรับฐาน ดีดขึ้นเล็กน้อย
หนุนจิตวิทยาการลงทุนวันนี้ ในทางเทคนิค SET ยังคงพักตัวอยู่เหนือเส้นค่าเฉลี่ย 50 และ 200 วันที่ 1560 และ 1550 จุดยังคงอยู่ในขาขึ้น
กลยุทธ์การลงทุน  หาก SET ยังถอยลงมาปิดไม่ต่ำกว่า 1565-1550 จุด แนะนำ อ่อนตัวซื้อ หรือ Selective Buy ในกลุ่มอุตสาหกรรมหรือหุ้นที่มีปัจจัยหนุนเฉพาะตัว จากมุมมองขาขึ้นในระยะกลางจากประเด็นการประชุม กนง. ปลายเดือนนี้
