โบรกฯ 2 คาด ตลาดหุ้นวันนี้ขึ้นลงในกรอบแคบๆ คาดการณ์ทิศทางได้ยาก ในทางเทคนิคดัชนีฯ จะเคลื่อนไหวลงกรอบ 1495-1515 จุด แม้วานนี้ปิดแดนลบ แต่แรงขายไม่น่าจะรุนแรงต่อเนื่องในวันนี้มากนัก เพราะสถาบันและต่างชาติยังคงซื้อสุทธิ ประเด็นที่ต้องติดตามใกล้ชิดคือ การไหลอออกของเงินทุนต่างชาติหลังบาทอ่อนค่า และ MSCI ปรับพอร์ตดัชนี
บล.ธนชาต  คาดแนวโน้ม SET ระยะสั้นเคลื่อนไหวขึ้นลงในกรอบแคบๆ (Sideways) บริเวณ 1485 – 1520 จุด โดยยังคงต้องติดตามการไหลออกของเงิน จากการอ่อนตัวของค่าเงินบาทต่อไป แนะนำ “Selective Buy”
หุ้นที่ผลการดำเนินงาน 1Q15 ออกมาดีอย่าง INTUCH THCOM
กลุ่มหุ้นส่งออกที่ได้ประโยชน์จากการอ่อนค่าของเงินบาทอย่าง GFPT SAPPE TUF SVI และ HANA
รวมถึงหุ้นที่ได้รับผลดีจากอัตราดอกเบี้ยที่ลดลงอย่าง INTUCH ADVANC และ TISCO
ในทางเทคนิค  พิจารณาจากกราฟรายวันพบว่า SET อยู่ต่ำกว่าเส้นค่าเฉลี่ยระยะสั้น 7-14-21 วัน ที่เคลื่อนไหวในทิศทางลง รวมถึงดัชนีเคลื่อนไหวอยู่ในลักษณะช่องทางแนวโน้มขาลง เป็นความเสี่ยงระยะกลางที่ดัชนี SET จะลงต่ำกว่า 1480-85 จุด ด้วยเป้าหมายบริเวณ 1430-40 จุด สำหรับวันนี้คาดว่าดัชนี SET จะเคลื่อนไหวลงในกรอบ 1495-1515 จุด

บล.เคเคเทรด  คาดว่าวันนี้ SET จะแกว่งตัวออกข้าง ในกรอบ 1495 – 1510 จุด สำหรับสัญญาณเชิงลบจากการพักตัววานนี้ เราเชื่อว่าไม่รุนแรงมาก เพราะ
1) แรงขายสุทธิหลักมาจากบัญชี บล.(Prop traders)ราว 2 พันล้านบาท แต่กลุ่มนักลงุทนสถาบันในประเทศและนักลงทุนต่างชาติ ยังซื้อสุทธิเข้ามามากถึง 1.8 พันล้านบาท และ 1.7 พันล้านบาท ตามลำดับ และ
2)มูลค่าการซื้อขายลดลงเหลือ 3.2 หมื่นล้านบาท ต่ำสุดตั้งแต่ 10 เม.ย.58 ที่ซื้อขายราว 3 หมื่นล้านบาท MSCI จะประกาศผลการปรับพอร์ตดัชนี ในวันที่ 12 พ.ค.(เราจะทราบผลในช่วงเช้ามืดของวันพุธที่ 13 พ.ค.)
และมีผลบังคับใช้ในวันที่ 1 มิ.ย.(Effective date) ตามสถิติย้อนหลัง 7 ปี (ปี 2551 – 2557) พบว่า หุ้นไทยที่ถูกคัดเลือกเข้าดัชนี Global Standard มักปรับตัวเพิ่มขึ้นโดยเฉลี่ย 9.8% ตั้งแต่วันประกาศผลจนถึงวัน Effective date ซึ่งกินระยะเวลาราว 2 สัปดาห์ ส่วนหุ้นเข้าดัชนี Global Small Cap ปรับตัวเพิ่มขึ้นโดยเฉลี่ยเพียง 1.1% ในช่วงเวลาเดียวกัน ถือเป็นโอกาสในการเก็งกำไรบริษัทที่ประกาศงบการเงิน 1Q58
