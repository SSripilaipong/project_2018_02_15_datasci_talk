โบรกฯคาดหุ้นไทยวันนี้ปรับลงต่อเนื่อง คาดนักลงทุนรอผลการลงประชามติของกรีซ แนะเลือกหุ้นเป็นรายตัว เน้น story และพื้นฐานดี ปันผลสูงและควรทยอยขายบ้างเมื่อมีกำไร

บริษัทหลักทรัพย์ เคเคเทรด จำกัด เปิดเผยว่า แนวโน้มกลุ่มธนาคารยังคงกดดัน SET ต่อเนื่อง โดยวานนี้(2 ก.ค.) กลุ่มธนาคารปรับตัวลดลงแรงกว่า 3.1% เป็นที่น่าสังเกตว่ามูลค่าการซื้อขายของ SET เพิ่มเป็น 4.7 หมื่นล้านบาท หนุนด้วยมูลค่าการซื้อขายของธนาคาร 3 แห่ง(KBANK, SCB, KTB) รวมกัน 1.2 หมื่นล้านบาท
สำหรับวันนี้(3 ก.ค.) บล.เคเคเทรด ประเมินว่า SET ทรงตัว และอาจจะพักตัวเล็กน้อย โดยให้แนวรับไว้ที่ระดับ 1485 จุดและแนวต้านที่ระดับ 1500 จุด และเชื่อว่า แรงขายกลุ่มธนาคารน่าจะเบาบางลงในระยะสั้นและไม่กดดัน SET มากในวันนี้
ขณะที่ บริษัทหลักทรัพย์ยูโอบี เคย์เฮียน (ประเทศไทย) (UOBKH) คาดแนวโน้มตลาดวันนี้มีโอกาสปรับลงอีกวัน โดยนักลงทุนรอผลการลงประชามติของกรีซในวันอาทิตย์ที่ 5 ก.ค.นี้ และความกังวลเรื่องหนี้เสียของกลุ่มแบงก์ยังมีอยู่ อาจส่งผลให้มีแรงขายของกลุ่มแบงก์ต่อเนื่องประกอบกับราคาน้ำมันดิบปรับลงมาแรงอย่างต่อเนื่องในสัปดาห์นี้เป็นอีกหนึ่งปัจจัยที่กดดันหุ้นกลุ่มพลังงานและ SET Index ส่วนทาง
ด้านตัวเลขเศรษฐกิจของสหรัฐที่ออกมาในสัปดาห์นี้ ถึงแม้ว่าการจ้างงานจะเพิ่มขึ้นต่อเนื่อง แต่การปรับขึ้นของรายได้ก็ต่ำ กว่าที่นักเศรษฐศาสตร์คาดไว้ วันนี้ตลาดไทยจึงน่าจะผันผวนเชิงลบ
โดยให้แนวรับที่ 1490 จุด แนวต้าน 1520 จุด สัดส่วนการลงทุน เงินสด 50% พอร์ตหุ้น 50%
กลยุทธ์ เรามองประเด็นกรีซจะทำให้หุ้นไทยขยับลงชั่วคราวและเป็นโอกาสดีในการซื้อหุ้น เลือกหุ้นเป็นรายตัว เน้น story และพื้นฐานดี ปันผลสูงและควรทยอยขายบ้างเมื่อมีกำไร
ส่วนค่าเงินบาทวันนี้เปิดตลาดที่ 33.77-33.70 บาท/ดอลล์ แข็งค่าเล็กน้อย จากปิดตลาดวานนี้ หลังเงินดอลลาร์อ่อนค่า จากตัวเลขจ้างงานนอกภาคเกษตรเดือนมิ.ย. เพิ่มขึ้นน้อยกว่าที่นักวิเคราะห์คาดการณ์ คาดแนวโน้มค่าเงินบาทวันนี้จะเคลื่อนไหวในกรอบ 33.70-33.80 บาท/ดอลล์
บล.ธนชาต แนะนำ “ซื้อ” กลุ่มรับเหมาฯ และกลุ่มหุ้นปันผลสูง อีกทั้งคาดงบไตรมาส 2/58 ออกมาดี อย่าง CK ,STEC ,SEAFCO ,INTUCH ,TRUEIF ,PTTGC ,BCP ,BLA
บล.เคจีไอ (ประเทศไทย) ระบุหุ้นเด่นวันนี้ ตามปัจจัยพื้นฐาน หากรับเสี่ยงได้แนะนำเก็งกำไร PLANB อีกทั้งแนะนำสะสมหุ้นหลัก BANPU,SAMART,IFEC,INTUCH
