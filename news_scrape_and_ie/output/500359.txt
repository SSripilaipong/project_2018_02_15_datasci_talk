โบรกฯ คาด  ‘หุ้นไทย’  แนวโน้มระยะสั้นยังพักฐานต่อ เผยหุ้นรับเหมาฯ เด่นน่าลงทุนวันนี้
บล.ธนชาต  ระบุในบทวิเคราะห์วันนี้ (17 มิ.ย.) ว่า   การทำประชามติของอังกฤษวันที่ 23 มิ.ย.นี้ กดดันตลาดหุ้นโลก และ ตลาดหุ้นไทย โดยมองว่าแนวโน้มระยะสั้นยังอยู่ในช่วงการพักฐานต่อไป จนกว่าจะมีความชัดเจนจากกรณี Brexit ล่าสุดโพล 45% ออกจากอียู และ 42% ยังอยู่ต่อ ขณะที่ 13% ยังไม่ตัดสินใจ ทั้งผลกระทบทางพื้นฐานต่อเศรษฐกิจไทยจำกัดโดยไทยส่งออกไปอังกฤษ 2% ของส่งออกทั้งหมด, นักท่องเที่ยวจากอังกฤษคิดเป็น 3% ของท่องเที่ยวทั้งหมด และ FDI จากอังกฤษน้อยกว่า 1%
สำหรับกลยุทธ์การลงทุนมองการบริโภคภายในประเทศฟื้นตัว, รับเหมาฯ เร่งลงทุนโครงสร้างพื้นฐาน ดังนี้ 1) รับเหมาฯ มีการเร่งลงทุนโครงสร้างพื้นฐาน ได้แก่ CK, SEAFCO 2) Asset Funds ซื้อ JASIF, DIF CPNRF 3) Consumption Plays ซื้อ ROBINS, BEAUTY, THANI, KTC 4) เก็งกำไร THAI ด้วยแนวต้าน 18.7/19.5 บาท ได้ประโยชน์น้ำมันลง

นักวิเคราะห์ มอง  ‘หุ้นไทย’  ยังกังวลต่อกรณี Brexit คาด SET แกว่งตัวในกรอบ 1,410-1,440 จุด
บล.บัวหลวง  ระบุในบทวิเคราะห์วันนี้ (16 มิ.ย.) ว่า ตลาดหุ้นไทยยังมีความกังวลเกี่ยวกับความผันผวนในตลาดประเทศพัฒนาแล้ว ที่เพิ่มสูงขึ้นเนื่องจากตลาดประเทศหลักๆ สหรัฐฯ เยอรมัน และญี่ปุ่น สัปดาห์นี้ต่างปรับตัวลงแรงจากความกังวลที่สหราชอาณาจักรจะออกจากสหภาพยุโรป (Brexit) โดยผลการสำรวจล่าสุดเสียงข้างมากเลือกที่จะออก การทำประชามติจะจัดขึ้นในวันที่ 23 มิ.ย.นี้
นักลงทุนมองสถานการณ์ดังกล่าวว่า อาจปรับพอร์ตลดสัดส่วนการลงทุนในสินทรัพย์เสี่ยงลงก่อนถึงวันทำประชามติตลาดหุ้นไทยก็อาจได้รับผลกระทบได้เช่นกัน ดังนั้นปัจจัยภายนอกในระยะนี้ดูเหมือนจะส่งผลต่อทิศทางของตลาดหุ้นไทยมากกว่าสัญญาณทางเทคนิคของตลาด สินทรัพย์เสี่ยงกำลังอยู่ภายใต้แรงกดดัน ดัชนีตลาดหุ้นสหรัฐ S&P500 และยุโรปกำลังปรับตัวลงสวนทางกับราคาทองที่ปรับตัวเพิ่มขึ้นมาจากการหวาดกลัวการลงทุนในสินทรัพย์เสี่ยง
กลยุทธ์การลงทุน เลือกลงทุนเป็นหุ้นรายตัว และใช้สัญญาณทางเทคนิคเข้ามาช่วยจับจังหวะการซื้อขาย แนะนำหลีกเลี่ยงหุ้นที่เกิดสัญญาณขายหรือการเปลี่ยนรูปแบบเป็นขาลง แนวโน้มตลาดสัปดาห์นี้ ยังคงเหมาะกับกลยุทธ์การเทรด เมื่อใดก็ตามที่ตลาดหุ้นส่งสัญญาณพักตัวลดความร้อนแรงลง หุ้นขนาดเล็ก-กลางหรือหุ้นที่ยังไม่ได้ปรับตัวขึ้นมากนักจะปรับตัวได้ดีกว่าขณะที่ภาพรวม SET แกว่งตัวอยู่ในกรอบที่ให้ไว้ 1410-1440 จุด

โบรกฯ คาด  ‘หุ้นไทย’  พักฐานต่อ เหตุกังวลมติ UK ออกจาก EU แนะ ซื้อหุ้นที่ได้รับผลดีจากดอกเบี้ยต่ำ
บล.ธนชาต  เปิดเผยว่า การแข็งค่าของค่าเงินดอลลาร์ฯ ก่อนการทำประชามติ UK ออกจากสมาชิก EU ในวันที่ 23 มิ.ย.นี้ กดดันราคาน้ำมัน และตลาดหุ้นโลก และคาดว่าจะส่งผลให้ SET อยู่ในช่วงการพักฐานต่อไป ขณะที่การประชุม FOMC คืนนี้ คาดว่าจะคงดอกเบี้ยที่ 0.25-0.50% และยังไม่ขึ้นดอกเบี้ยในการประชุมเดือน ก.ค.นี้ ขณะที่การประชุม BOJ ในวันที่ 16 มิ.ย.นี้อาจออกมาตรการกระตุ้นเศรษฐกิจเพิ่มเติม
สำหรับกลยุทธ์การลงทุน แนะซื้อ Consumption Plays ตามแนวโน้มการบริโภคฟื้นตัว และกลุ่มหุ้นที่ได้รับผลดีจากแนวโน้มดอกเบี้ยต่ำยาวนานอย่าง Infrastructure Fund 1) Consumption Plays ตามแนวโน้มเศรษฐกิจฟื้นตัวแนะซื้อ BEAUTY, GLOBAL, THANI (ธุรกิจปล่อยสินเชื่อรถบรรทุกฟื้นตัว ต้นทุนการเงินต่ำ และปันผลสูง 5-6%) KTC และ เก็งกำไร CPALL 2) กลุ่มที่ได้รับผลดีจากดอกเบี้ยต่ำ คือ JASIF (ปันผล 9-10%) BTSGIF และ DIF

โบรกฯ มอง ตลาด  ‘หุ้นไทย’  กังวลการลงมติของอังกฤษที่จะออกจาก EU หรือไม่ เปิดโผ! หุ้นเด่นน่าลงทุนรับผลดีจากดอกเบี้ยต่ำ
สำหรับตลาดหุ้นไทยช่วงนี้ มีความกังวล Brexit คือในวันที่ 23 มิ.ย.นี้ อังกฤษจะมีการลงประชามติว่าจะออกจากสหภาพยุโรป (EU) หรือไม่ อีกทั้งยังกังวลในการปรับลดลงของราคาน้ำมันกดดันตลาดหุ้นโลก โดย 2 ประเด็นดังกล่าวกดดันตลาดหุ้นโลกต่อเนื่อง ทั้ง Dow Jones -0.74% และ ตลาดหุ้นยุโรป -1-2% เมื่อคืนนี้ ขณะที่ราคาน้ำมันยังทรงตัว
ทั้งนี้ คาดว่า SET อยู่ในช่วงการพักฐานต่อเนื่อง ด้วยแนวรับ 1,412 จุด ที่เคยเปิด Gap ทางเทคนิคไว้ในช่วงปลายเดือน พ.ค.ที่ผ่านมา โดยกลุ่มหุ้นที่ได้รับผลดีจากดอกเบี้ยต่ำ+ปันผลสูง ได้แก่ REIT, Infra-Fund ขณะที่ กลุ่มเช่าซื้อ มีแนวโน้มแข็งแกร่งกว่าตลาดช่วงนี้ แนะซื้อกลุ่มหุ้นที่ได้รับผลดีจากดอกเบี้ยต่ำ
รวมถึงซื้อกลุ่มโรงพยาบาลซึ่งมีแนวโน้มแกร่งกว่าตลาด ได้แก่ 1) กลุ่มหุ้นที่ได้รับผลดีดอกเบี้ยต่ำ ปันผลสูง คือ JASIF, THANI (กำไรโตสูง 35-19% ในปี 2016-17 ปันผล 5-6.5%) 2) กลุ่มโรงพยาบาลดูแข็งแกร่งกว่าตลาด ซื้อ CHG (กำไรโตสูงสุดในกลุ่ม 23-27% ในปี 2016-17 ทางเทคนิคทดสอบ 3.20 บาท – ดู Trade Code ประกอบ) และ เก็งกำไร BCH (ต้าน 12.7/13.1 บาท)

คาด!  ‘หุ้นไทย’  ลงตามภูมิภาค เผย หุ้นน่าลงทุนช่วงดัชนีผันผวน แนะ รอสะสมกลุ่มที่อิงกับเศรษฐกิจในประเทศ
บล.บัวหลวง  ระบุ หุ้นไทยวันนี้ (13มิ.ย.) คาด ดัชนี SET ลงตามภูมิภาค โดยจะปรับฐานลงทดสอบ แนวรับบริเวณ 1,415-1,400 จุด กลยุทธ์ แนะกระชับพอร์ต เพื่อรอประเมินสถานการณ์ใหม่ ว่าดัชนีจะปรับฐานลงไปมากแค่ไหน ทั้งนี้พัฒนาการจะขึ้นอยู่กับสถานการณ์ตลาดโลก ได้แก่ โพล การลงประชามติอังกฤษ ว่าจะอยู่ หรือ ออกจากการเป็นสมาชิกยูโร, สุนทรพจน์ ของเจ้าหน้าที่ ECB ตลอดสัปดาห์นี้ ซึ่ง ตลาดหวังจะได้เห็น ท่าทีต่อมาตรการรองรับกรณี อังกฤษออกจากยูโรโซน, ถ้อยแถลงประธานเฟดในการประชุมวันพุธ ตลาดคาดคงดอกเบี้ย เพื่อรอประเมินสถานการณ์ยุโรป
